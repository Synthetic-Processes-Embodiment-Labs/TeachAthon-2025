## 🚀 Project Description
StudyAI is a course-specific AI tutoring platform that turns uploaded syllabi, lecture slides, readings, and assignments into an evolving knowledge base for personalised Socratic tutoring. It adapts explanations and practice to each student, tracks misconceptions over time, and uses separate AI modules to generate, verify, and refine its responses.
