## 🚀 Project Description
Symbol Sense is a web app that analyses uploaded images of mathematical notation and equations, explaining the symbols, variable relationships, and derivation steps in context. Students can then ask follow-up questions, generate practice problems, and access related learning resources through its chat interface.
