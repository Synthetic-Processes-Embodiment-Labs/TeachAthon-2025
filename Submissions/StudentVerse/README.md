## 🚀 Project Description
StudentVerse is a gamified student productivity and learning platform that uses AI to turn schedules, deadlines, and personal goals into manageable daily micro-tasks, rewarding completion through an evolving virtual universe. It also generates prerequisite roadmaps, provides coursework feedback, and supports mock-exam practice.
