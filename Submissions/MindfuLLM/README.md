## 🚀 Project Description
MindfuLLM is a Chrome extension that turns selected academic conversations from ChatGPT or ELM into personalised multiple-choice and free-response quizzes. It sends the generated quiz to the student as an editable Google Form, allowing them to test and revisit what they learned through the LLM.
